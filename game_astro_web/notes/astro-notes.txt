1. Astro is a JavaScript Framework - focussed on performance.
	- JavaScript is a programming language made to run in the browser.
    - a JavaScript framework lets us use js to help us build html and css more efficiently.

2. When we write Astro code, it will be built into a bundle of HTML, CSS and JS.
    - html is for adding text, images, etc to our webpage - elements and structure.
    - css is for stlying the html elements - can also use it for layouts.
	- JavaScript lets us run code in the browser that can interact with our webpage - this means
		it can update the html and css live on the browser in response to user interaction.

3. Project structure:
	src/ - all of the code we write goes in here.
	src/assets - contains images and fonts etc that we need for our site.
    src/components - stores reusable astro components.
    src/layouts - stores a reusable page layout wrapper for our website.
    src/pages - every file in here is a url endpoint - i.e. a page on our site.
	e.g. 
		https://dupagames.com/ 	-> 	index.html (root)
		https://dupagames.com/games/chess
		https://dupagames.com/games/space_shooter
	src/styles - stores the global css files that apply to every page 
		- we can also write scoped (local) styles inside .astro files
