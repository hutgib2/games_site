my html(stands for HyperText Markup Language) defintion paragraph: Html is a standard markup language that makes up the web page's layout. 
HTML can be used to style, layout or resize basic text content and media files for example.
html specifies what tags are put in or not.

After that, below there is a list of html tags which will cover 99% of the tags we will use on our site. For each one I must write, it's:

- name
- semantic meaning
- use/functionality


e.g.

<div>
- name: division
- semantic meaning: generic grouping tag
- use: used to group chunks of html with no clear semantic connection, often used to apply styling (CSS) or interactivity (js) to a group of elements*

*html "tag" and html "element" are interchangeable words 

HTML tags:
<a>
- name: anchor element.
- semantic meaning: a hyperlink that connects to other files or web pages.
- use/functionality: to have links inside web pages.

<button>
- name: button tag.
- semantic meaning: a clickable button that triggers an action.
- use/functionality: it's used to have buttons on your page.

<header>
- name: header element.
- semantic meaning: group introductory content.
- use/functionality: the header element is used for grouping parts of the text that are apart of the web page's header.

<footer>
- name: footer element.
- semantic meaning: group ending content.
- use/functionality: the footer element is used for grouping parts of the text that are apart of the web page's bottom section.

<section>
- name: section.
- semantic meaning: adds one singular thematic heading to a group of content.
- use/functionality: groups different parts of a page all together.

<article>
- name: article tag.
- semantic meaning: an standalone and self-contained composition.
- use/functionality: to be redistributed onto different pages.

<ol>
- name: ordered list.
- semantic meaning: a numbered list.
- use/functionality: can be used to put text in numbered rows.

<ul>
- name: unordered list.
- semantic meaning: a non-numeric list.
- use/functionality: can be used to put text in bullet points.

<li>
- name: list item.
- semantic meaning: an item in a list.
- use/functionality: can be used for each item of a ordered or unordered list.

<p>
- name: paragraph element.
- semantic meaning: a reprensation of a particular block of text.
- use/functionality: to seperate paragraphs.

<h1..6>
- name: heading elements 1 to 6.
- semantic meaning: levels of section headings where h1 has the biggest text size while h6 has the smallest text size.
- use/functionality: to present text as a title or a subsection for example.

<iframe>
- name: inline frame
- semantic meaning: a window that adds seperate web docs inside a main page.
- use/functionality: for putting a webpage in a webpage

<nav>
- name: navigation section element
- semantic meaning: a section for major navigation links.
- use/functionality: to help a user navigate pages or the website itself.

Each element can also be given a number of "attributes" - extra information that allow us to assign names/data to elements and/or groups of elements.
